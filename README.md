# Painel RH GPV

Painel interno para gestão de vagas e candidaturas do Portal de Carreiras GPV.

## Vercel
Use as mesmas variáveis do portal público:
- GAS_WEB_APP_URL
- GAS_PROXY_SECRET

## Antes do primeiro acesso
Atualize o Google Apps Script com o pacote `Apps-Script-RH-GPV-admin`, execute `setupProject()`, configure o primeiro usuário em `criarUsuarioAdminInicial()` e atualize a implantação do Web App.
